# Лендинг “Детский массажист Оксана”

Небольшой одностраничный сайт с зелёными акцентами, навигацией, формой записи и простым функционалом на JavaScript.

## Структура

- `index.html` — страница
- `css/style.css` — стили
- `js/main.js` — функционал (скролл к форме, валидация, модальное окно, сохранение заявки)
- `images/` — изображения (`oksana.png`)

## Запуск локально

Вариант 1 (самый простой): откройте `index.html` двойным кликом.

Вариант 2 (как “мини‑проект” через локальный сервер):

1. Установите Node.js (если ещё не установлен).
2. В папке проекта выполните:

```bash
npm run dev
```

Сервер поднимется и вы сможете открыть сайт в браузере по адресу, который покажет команда.

## Где лежат заявки

После отправки формы данные сохраняются в браузере (в `localStorage`) под ключом `oksana_leads`.
